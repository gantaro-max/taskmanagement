# 機能一覧

## アプリケーション概要

Trello風のタスク管理Webアプリ。ボード・リスト・カードの3階層でタスクを管理する。
学習目的のシングルユーザー向けローカルアプリ（マルチユーザー・認証機能なし）。

---

## 機能一覧

### ボード管理

| # | 機能 | 概要 |
|---|-----|------|
| B-1 | ボード一覧表示 | 登録されているボードをカード形式で一覧表示する |
| B-2 | ボード作成 | ボード名を入力して新しいボードを作成する |
| B-3 | ボード削除 | 確認後にボードを削除する（配下のリスト・カードも削除） |

### リスト管理

| # | 機能 | 概要 |
|---|-----|------|
| L-1 | リスト表示 | ボード内のリストをカラム形式で横並びに表示する |
| L-2 | リスト作成 | リスト名を入力してボードにリストを追加する |
| L-3 | リスト名編集 | リスト名をインライン編集する |
| L-4 | リスト削除 | リストを削除する（配下のカードも削除） |

### カード管理

| # | 機能 | 概要 |
|---|-----|------|
| C-1 | カード表示 | リスト内のカードを縦並びに表示する |
| C-2 | カード作成 | タイトルを入力してリストにカードを追加する |
| C-3 | カード詳細表示・編集 | カードのタイトルと説明文をモーダルで表示・編集する |
| C-4 | カード削除 | カードを削除する |
| C-5 | カード移動（D&D） | ドラッグ＆ドロップでリスト間・リスト内を移動する |
| C-6 | カード移動（モバイル） | モバイル時はモーダル内のセレクトボックスでリストを変更する |

### UI・共通

| # | 機能 | 概要 |
|---|-----|------|
| U-1 | レスポンシブ対応 | デスクトップ（769px以上）とモバイル（768px以下）でレイアウトを切り替える |
| U-2 | 空状態表示 | ボードが0件のとき専用のメッセージと作成ボタンを表示する |
| U-3 | エラー通知 | API失敗時に画面右下にトースト通知を3秒表示する |
| U-4 | 削除確認 | ボード・カードの削除前に確認ダイアログを表示する |
