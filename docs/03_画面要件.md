# 画面要件

## 画面一覧

| 画面ID | 画面名 | 対応機能 |
|-------|-------|---------|
| S-01 | ボード一覧 | B-1, B-2, B-3 |
| S-01-a | ボード新規作成フォーム（S-01上に展開） | B-2 |
| S-01-b | ボード削除確認ダイアログ（S-01上に重なる） | B-3 |
| S-01-c | ボード一覧（空状態） | B-2 |
| S-02 | ボード詳細 | L-1〜L-4, C-1〜C-5 |
| S-02-a | カード追加インラインフォーム（S-02内に展開） | C-2 |
| S-02-b | リスト名インライン編集（S-02内） | L-3 |
| S-02-c | リスト追加フォーム（S-02右端に展開） | L-2 |
| S-02-d | カードドラッグ中状態（S-02内） | C-5 |
| S-03 | カード詳細モーダル（S-02上に重なる） | C-3, C-4, C-6 |
| S-M | モバイルレイアウト（768px以下） | 全機能 |
| S-E | エラートースト（全画面共通） | U-3 |

---

## S-01: ボード一覧

### 表示要件
- 登録されている全ボードをカード形式で表示する
- 各ボードカードに「タイトル」「カード総枚数」「削除ボタン」を表示する
- ボードが0件の場合は S-01-c（空状態）を表示する

### 操作要件
- ボードカードをクリック → S-02 へ遷移
- ヘッダーの `[+ 新規]` クリック → S-01-a を表示
- `[削除]` クリック → S-01-b を表示

---

## S-01-a: ボード新規作成フォーム

### 表示要件
- ボードカード一覧の末尾にインラインでフォームを展開する
- テキスト入力欄（自動フォーカス）・`[作成]` ボタン・`[×]` ボタンを表示する

### 操作要件
- `[作成]` または `Enter` → ボード作成 → フォームを閉じ一覧を更新
- `[×]` または `Escape` → フォームを閉じる（作成しない）
- タイトル未入力で送信 → 入力欄を赤枠にしてエラーメッセージを表示する

---

## S-01-b: ボード削除確認ダイアログ

### 表示要件
- 背景を暗転して中央にダイアログを表示する
- 削除対象のボード名・警告文（配下のデータもすべて削除される旨）を表示する
- `[削除する]` ボタンと `[キャンセル]` ボタンを表示する

### 操作要件
- `[削除する]` → ボードを削除 → ダイアログを閉じ一覧を更新
- `[キャンセル]` または背景クリック → ダイアログを閉じる（削除しない）

---

## S-01-c: ボード一覧（空状態）

### 表示要件
- アイコン・「ボードがまだありません」のメッセージ・作成ボタンを中央に表示する

### 操作要件
- `[+ 最初のボードを作成する]` クリック → S-01-a と同じ動作

---

## S-02: ボード詳細

### 表示要件
- ヘッダーに「← 一覧に戻る」「ボード名」「`[+ リスト追加]`」を表示する
- リストを `position` 昇順で横並びに表示する（デスクトップ）
- 各リストにリスト名・編集ボタン `[✎]`・カード一覧・`[+ カード追加]` を表示する
- 各カードにタイトルと削除ボタン `[×]` を表示する

### 操作要件
- `← 一覧に戻る` → S-01 へ遷移
- カードクリック → S-03 を表示
- `[×]` クリック → カードを即削除（確認なし）
- `[+ カード追加]` クリック → S-02-a を展開
- リスト名の `[✎]` クリック → S-02-b に切り替え
- `[+ リスト追加]` クリック → S-02-c を展開
- カードをドラッグ開始 → S-02-d（ドラッグ中状態）へ

---

## S-02-a: カード追加インラインフォーム

### 表示要件
- クリックされたリストの `[+ カード追加]` ボタンをフォームに置き換えて表示する
- テキスト入力欄（自動フォーカス）・`[追加]` ボタン・`[×]` ボタンを表示する

### 操作要件
- `[追加]` または `Enter` → カード作成 → フォームをリセット（連続入力可能）
- `[×]` または `Escape` → フォームを閉じる
- タイトル未入力で送信 → 入力欄を赤枠にしてエラー表示

---

## S-02-b: リスト名インライン編集

### 表示要件
- リストのタイトルテキストをテキスト入力欄に切り替える（既存テキストを選択済みで表示）
- 確定ボタン `[✓]` を表示する

### 操作要件
- `[✓]` または `Enter` → リスト名を更新してテキスト表示に戻る
- `Escape` → 変更を破棄してテキスト表示に戻る
- タイトル未入力での保存は不可

---

## S-02-c: リスト追加フォーム

### 表示要件
- 既存リストの右端にカラム形式でフォームを表示する
- テキスト入力欄（自動フォーカス）・`[追加]` ボタン・`[×]` ボタンを表示する

### 操作要件
- `[追加]` または `Enter` → リスト作成 → フォームをリセット（連続入力可能）
- `[×]` または `Escape` → フォームを閉じる

---

## S-02-d: カードドラッグ中状態

### 表示要件
- ドラッグ中のカードは半透明・影付きで表示し、カーソルを `grabbing` にする
- 元の位置にはグレー枠の幽霊カードを表示する
- ドロップ可能なリスト全体の背景色をハイライト表示する
- カーソルが重なったリスト内に点線のドロップゾーンを表示する

---

## S-03: カード詳細モーダル

### 表示要件
- 背景を暗転して中央にモーダルを表示する
- タイトル入力欄・説明文テキストエリア（複数行）を表示する
- 所属リスト名・作成日・更新日を表示する
- モバイル時はリスト移動用のセレクトボックスを追加表示する
- `[保存]` `[削除]` `[閉じる]` ボタンを表示する

### 操作要件
- `[保存]` または `Ctrl+Enter` → カード情報を更新 → モーダルを閉じる
- `[削除]` → 確認ダイアログ → カード削除 → モーダルを閉じる
- `[閉じる]` または背景クリック → 変更を破棄してモーダルを閉じる

---

## S-M: モバイルレイアウト（768px以下）

### レイアウト変更
| 要素 | デスクトップ | モバイル |
|-----|------------|---------|
| ボード一覧 | カードグリッド | カード縦積み（全幅） |
| ボード詳細リスト | 横並び（スクロール） | 縦積み（全幅） |
| カードD&D | 有効 | 無効（S-03のセレクトで代替） |

---

## S-E: エラートースト（全画面共通）

### 表示要件
- 画面右下に固定表示する
- 種別（エラー: 赤 / 成功: 緑）・メッセージ・`[×]` ボタンを表示する

### 動作要件
- 3秒後に自動消去する
- `[×]` クリックで即時消去する
- API通信エラー時に必ず表示する
