## S-02-d: カードドラッグ中状態

```
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│  │ ToDo    [✎] │  │ Doing   [✎] │  │ Done    [✎] │
│  ├──────────────┤  ├──────────────┤  ├──────────────┤
│  │ ┌──────────┐ │  │ ┌──────────┐ │  │              │
│  │ │░░░░░░░░░░│ │  │ │ 画面設計  │ │  │ ┌ ─ ─ ─ ─ ┐ │  ← ドロップゾーン
│  │ │░ 要件定義 │ │  │ │      [×] │ │  │   （点線）   │
│  │ │░(ドラッグ)│ │  │ └──────────┘ │  │ └ ─ ─ ─ ─ ┘ │
│  │ │░░░░░░░░░░│ │  │              │  │              │
│  │ └──────────┘ │  │              │  │              │
│  │ ┌ ─ ─ ─ ─ ┐ │  │              │  │              │
│  │   （幽霊）   │  │              │  │              │
│  │ └ ─ ─ ─ ─ ┘ │  │              │  │              │
```

| 要素 | 見た目 |
|-----|--------|
| ドラッグ中のカード | 半透明・影付き・カーソルが `grabbing` |
| 元の位置（幽霊） | 薄いグレーの枠のみ残す |
| ドロップゾーン | 点線ボーダー・背景色を薄く変更 |
| ドロップ可能なリスト | 背景色をハイライト |

## 画面遷移図

```mermaid
stateDiagram-v2
    [*] --> S01: アプリ起動

    state "S-01 ボード一覧" as S01
    state "S-01-a 新規作成フォーム" as S01a
    state "S-01-b 削除確認" as S01b
    state "S-01-c 空状態" as S01c
    state "S-02 ボード詳細" as S02
    state "S-02-a カード追加フォーム" as S02a
    state "S-02-b リスト名編集" as S02b
    state "S-02-c リスト追加フォーム" as S02c
    state "S-03 カード詳細モーダル" as S03

    S01c --> S01a: [+ 最初のボードを作成する]
    S01  --> S01a: [+ 新規]
    S01a --> S01 : 作成 / キャンセル

    S01  --> S01b: [削除]
    S01b --> S01 : 削除確定 / キャンセル

    S01  --> S02 : ボードカードをクリック
    S02  --> S01 : ← 一覧に戻る

    S02  --> S02a: [+ カード追加]
    S02a --> S02 : 追加 / キャンセル

    S02  --> S02b: リスト名[✎]
    S02b --> S02 : 確定 / Escape

    S02  --> S02c: [+ リスト追加]
    S02c --> S02 : 追加 / キャンセル

    S02  --> S03 : カードをクリック
    S03  --> S02 : 保存 / 閉じる / 削除
```
